# PetQuirky Docs Index

本目录是 PetQuirky 的项目文档中心。外层 `/website` 只作为工作区；长期有效的项目文档都应放在本仓库的 `docs/` 下。

## Directory Map

| Directory | Purpose |
|---|---|
| `00-project-context/` | 项目背景、阶段总结、架构、多角色协作体系、历史执行指令 |
| `01-launch/` | 上线清单、自动化测试计划、集成测试记录模板、发布前 QA |
| `02-product-research/` | 选品、供应商、样品、产品评分、利润测算 |
| `03-operations/` | TikTok、SEO、内容日历、运营活动、转化策略 |
| `04-weekly/` | 周计划、周报、复盘 |
| `05-feature-notes/` | 功能改版记录、设计调整、专项说明 |

## Writing Rules

1. 新增长期文档时优先放进对应分类目录。
2. 不再把 PetQuirky 项目文档放到仓库外层 `/website`。
3. 周计划和周复盘统一放在 `04-weekly/`。
4. 单个功能的改版说明放在 `05-feature-notes/`。
5. 外部平台账号、密钥、私密信息不要写入文档。
