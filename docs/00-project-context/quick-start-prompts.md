# PetQuirky Quick Start Prompts

你只需要记住一个入口：`皮蛋`。

## 最常用

```text
皮蛋：帮我制定本周计划
```

```text
皮蛋：帮我完成上线前准备
```

```text
皮蛋：帮我评估这个产品链接
```

```text
皮蛋：我不知道具体该卖哪个品，帮我做选品决策
```

```text
皮蛋：按猫狗引流、异宠利润的策略，帮我决定首批卖哪些产品
```

```text
皮蛋：给这个产品做 7 天 TikTok 内容
```

```text
皮蛋：帮我检查网站还有没有上线阻断问题
```

```text
皮蛋：这是本周数据，帮我复盘
```
